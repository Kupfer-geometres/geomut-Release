# GeoMut - Dépôt de Releases

Ce dépôt contient les versions compilées du plugin QGIS **GeoMut**.

## Installation via le dépôt QGIS custom

1. Dans QGIS, aller dans **Extensions → Installer/Gérer les extensions**
2. Onglet **Paramètres**
3. Cliquer sur **Ajouter** dans la section "Dépôts de plugins"
4. Remplir :
   - **Nom** : GeoMut Release
   - **URL** : `https://raw.githubusercontent.com/Kupfer-geometres/geomut-Release/main/plugins.xml`
5. Cliquer sur **OK**
6. Onglet **Tout**, chercher "GeoMut" et l'installer

## Installation manuelle

Télécharger le fichier `.zip` depuis la [page des releases](https://github.com/Kupfer-geometres/geomut-Release/releases) et l'installer via **Extensions → Installer/Gérer les extensions → Installer depuis un ZIP**.

## Mises à jour

Si vous avez ajouté le dépôt custom, QGIS détectera automatiquement les nouvelles versions disponibles.
